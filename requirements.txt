INSTALLATION NOTES

============================================================================

1. Create virtual environment:

python3 -m venv venv



2. Activate virtual environment:

- Linux/Mac: source venv/bin/activate

- Windows: venv\Scripts\activate



3. Install dependencies:

pip install -r requirements.txt



4. Verify installation:

python -c "import paramiko, cryptography, psutil, colorama, requests, dns.resolver; print('OK')"

============================================================================

SECURITY NOTES

============================================================================

IMPORTANT:

- Always use virtual environment

- Keep dependencies updated regularly

- Review security advisories

- Pin versions in production

pip install pip-audit

pip-audit

To update all packages:

pip install --upgrade -r requirements.txt

============================================================================

TROUBLESHOOTING

============================================================================

If you encounter installation errors:



1. Update pip:

pip install --upgrade pip setuptools wheel



2. Install build dependencies (Linux):

sudo apt-get install python3-dev build-essential libssl-dev libffi-dev



3. Install build dependencies (CentOS/RHEL):

sudo yum install python3-devel gcc gcc-c++ openssl-devel libffi-devel



4. For specific package errors, install individually:

pip install paramiko

pip install cryptography

etc.
